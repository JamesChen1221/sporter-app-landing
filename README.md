# Sporter Landing Page

這是一個運動社交 app 的引導網頁專案。Sporter 是一個運動社交平台，讓用戶可以發起活動、尋找運動夥伴、分享運動動態，是一個我正在嘗試的 side project。

This is the landing page project for Sporter, a sports social app. Sporter is a platform where users can create activities, find sports partners, and share their sports moments — a side project I'm currently building.

---

## 專案目的 / Purpose

提供 Sporter app 的對外展示頁面，讓訪客了解平台的核心功能與特色，並引導用戶前往 app 入口。

The landing page serves as the public-facing introduction to the Sporter app, giving visitors an overview of its core features and directing them to the app.

---

## 關於 Sporter / About Sporter

Sporter 是一個以「找人一起運動」為核心的社交平台，目標是降低運動的門檻——不需要固定球友，不需要提前認識，只要發起或加入一場活動，就能找到志同道合的運動夥伴。

Sporter is a social platform built around the idea of finding people to play sports with. The goal is to lower the barrier to exercise — you don't need a regular group or prior connections. Just create or join an activity, and you'll find like-minded partners ready to play.

### 核心功能 / Core Features

**🏃 活動配對 / Activity Matching**
用戶可以發起運動活動，設定運動類型、時間、地點與人數。其他用戶加入後，人數達到門檻即自動確定，整個流程不需要額外溝通。支援羽球、籃球、足球、網球、排球、桌球、匹克球等多種運動。

Users can create sports activities by setting the sport type, time, location, and player count. Once enough participants join, the activity is automatically confirmed — no extra coordination needed. Supported sports include badminton, basketball, football, tennis, volleyball, table tennis, pickleball, and more.

**📍 附近搜尋 / Nearby Discovery**
根據用戶的所在位置，篩選指定距離範圍內的活動，讓用戶快速找到就近可以參與的運動機會。

Users can filter activities within a set distance from their current location, making it easy to find nearby sports opportunities.

**👥 社群動態 / Community Feed**
用戶可以發布運動相關的貼文、照片，與好友互動。精選動態會根據互動品質自動推薦優質內容。

Users can post sports-related updates and photos, and interact with friends. A curated feed surfaces high-quality content based on engagement.

**💬 即時聊天 / Real-time Chat**
支援用戶之間的一對一即時聊天，方便在活動前後保持聯繫。

One-on-one real-time messaging lets users stay in touch before and after activities.

**🌏 多語言支援 / Multilingual**
介面支援繁體中文、英文、印尼文、菲律賓文、越南文，面向東南亞多語言用戶群。

The interface supports Traditional Chinese, English, Indonesian, Filipino, and Vietnamese, targeting a multilingual Southeast Asian user base.

This project was built primarily with AI-assisted development. The design direction, feature planning, and content were driven by me, while AI (Kiro) was used extensively to help write code, debug issues, and refine details. It's part of my exploration into AI-collaborative development workflows.
